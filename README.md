# seegnature
EEG classification model using rough path signatures and a standard logistic regression classifier

# How to reproduce experiments
It is based on Notebook designed for Google colab.

The Notebook and the repo are self sufficient : Notebook can clone the repo and import the Seegnature package.

Notebook comparison_and_hyperparameters.ipynb contains the hyperparameters study and everything to reproduce all the tables except for LR+PSD and LR+BP which is in the basic_ML.ipynb.

Graphs.ipynb allow you to produces inter and intra graphs on electrodes. An interactive html figure is saved in the folder graphs for a larger view !
